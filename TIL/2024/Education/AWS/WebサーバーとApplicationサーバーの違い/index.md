# Web サーバーと Application サーバーの違い

- web サーバー
  - 静的コンテンツの生成を行う
    - html, css...
  - ウェブサイトのコードとデータをホストする技術
- application サーバー
  - api サーバーがここに含まれる
  - db との接続や動的コンテンツの生成を行う
